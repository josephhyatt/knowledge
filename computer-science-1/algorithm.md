# Algorithm

## Characteristics of an Algorithm

* **Input** - 0 or more
* **Output** - at least 1 output
* **Definiteness** -![](../.gitbook/assets/sqrt_-1.png)
* **Fineness**  
* **Effectiveness** 

## How to write an Algorithm

```cpp
Algorithm swap(a, b)
{
    temp = a;
    a = b;
    b = temp;
}
```
